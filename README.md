# Tic-Tac-Toe AI 🎮

## Description
This is a simple Tic-Tac-Toe game built using Python where a user plays against the computer (AI).

## Features
- User vs Computer game
- Detects win and draw
- Simple AI logic

## Technologies Used
- Python

## How to Run
1. Download the file
2. Open terminal
3. Run:
   python game.py

## Game Instructions
- Enter numbers from 1 to 9
- Try to beat the AI
